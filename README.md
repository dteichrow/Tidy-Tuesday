# Tidy Tuesday Practice Archive

Small archive of early R and Quarto data-cleaning practice.

This is not a flagship portfolio project. It is kept public as a lightweight
record of exploratory data work, package experimentation, and visualization
practice.

## Contents

- `2023-08-29-fair-use-index.qmd` explores the U.S. Copyright Office Fair Use Index,
  including table extraction, PDF parsing, text cleaning, and basic
  visualization experiments.
- `2023-09-05-union-wages.R` is a short R plotting fragment comparing union and
  non-union wage means over time.

## What It Demonstrates

- R/tidyverse data wrangling.
- Working with messy public data.
- Scraping and PDF/table extraction experiments.
- Iterative notebook-style exploration.

## Caveats

This repo contains exploratory notes, not production code. Some cells install
packages inline, some analysis is incomplete, and the files should be read as
practice material rather than a polished software project.

For professional review, the stronger public examples are the main Edge of
Epidemiology site and the Epi Dossier evidence pipeline.
